# Area-Insisghts

Built an end-to-end Area Insights dashboard integrating sales, rental, census, and crime datasets using Streamlit + MongoDB.

Engineered dynamic geo-based filtering (postcode, outcode, district) and automated aggregation pipelines for real-time KPIs.

Implemented modules for property pricing trends, bedroom/type analytics, agent performance, tax band distribution, crime severity, and Census 2021 demographics.

Developed robust functions for ETL, normalization, parsing, price history analysis, LSOA mapping, and custom visualizations using Plotly.

Enhanced UX with responsive UI, caching, and optimized query execution for 6,000+ records with multi-collection joins.
